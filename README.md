# CodeAlpha_FitnessTracker
A sleek and modern fitness tracking web app to help you monitor your daily workouts and progress.

## ✨ Features
- **Dashboard** with live stats: calories burned, workout time, number of workouts, and day streak
- **Daily Goals** with visual progress bars (calories, time, workouts)
- **Weekly Chart** showing calories burned across the last 7 days
- **Log Activity** form: choose exercise type, duration, calories, and add notes
- **History** tab showing all past activities with delete option
- Data is saved automatically using localStorage — no backend needed!

## 🛠️ Built With
- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage for data persistence

## 🚀 How to Run
1. Download `index.html`
2. Open it in any web browser — no installation needed!
